# Juegos Python

Este es un repo de juegos en python con la libreria Pygame.

Todavia me falta hacer que este proyecto de tetris este con uv para que las personas que lo prueben no tengan que hacer un:

```bash
 pip install pygame
 ```

Si no quieres instalarlo en tu equipo haz esto en tu terminal

1. Crea el espacio virtual
```bash
python -m venv .venv
```
2. Activa el `.venv`
```bash
.venv\Scripts\activate   # Windows
source .venv/bin/activate # macOS/Linux
```
3. Instala los paquetes desde `requirements.txt`
```bash
pip install -r requirements.txt
```
4. Para salir del entorno virtual es

```bash
deactivate
```
